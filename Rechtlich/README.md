# Vertrag-PDF Generator

Generates a print-ready PDF for the contract "Öffentlichkeitsarbeit Arztpraxis" using PDFKit.

## Usage
- Ensure Node.js 18+ is installed.
- Install dependencies: `npm install`
- Build PDF: `npm run build`
- Output: `out/Vertrag_Oeffentlichkeitsarbeit_Arztpraxis.pdf`

Adjust content in `src/contractContent.js` and styling/layout in `src/styles.js`.
