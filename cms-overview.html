Häufige Fragen

Kann ich ohne technische Kenntnisse das CMS verwenden?

Ja! Das CMS hat eine benutzerfreundliche Oberfläche. Sie benötigen nur eine einmalige Node.js-Installation, die Schritt-für-Schritt erklärt wird.

Welche Browser werden unterstützt?

Alle modernen Browser: Chrome, Firefox, Safari, Edge. Internet Explorer wird nicht unterstützt.

Wie sichere ich meine Daten?

Das CMS erstellt automatisch Backups. Sie können auch manuell Backups erstellen und herunterladen.

Ist das CMS sicher?

Ja, das CMS ist durch Passwort geschützt und läuft lokal auf Ihrem Server. Für öffentliche Server empfehlen wir zusätzliche Sicherheitsmaßnahmen.
