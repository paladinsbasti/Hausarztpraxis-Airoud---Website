CMS Setup

Hausarztpraxis Dr. Airoud - Content Management System

1

Node.js installieren

Laden Sie Node.js von der offiziellen Website herunter und installieren Sie es:

Node.js herunterladen

Empfohlene Version: LTS (Long Term Support)

Minimale Version: 14.0.0 oder höher

2

Terminal/Eingabeaufforderung öffnen

Öffnen Sie das Terminal (macOS/Linux) oder die Eingabeaufforderung (Windows) und navigieren Sie zu diesem Verzeichnis:

cd "Ihr/Pfad/zu/diesem/Ordner"
3

Abhängigkeiten installieren

Führen Sie den folgenden Befehl aus, um alle benötigten Pakete zu installieren:

npm install
4

Server starten

Starten Sie den CMS-Server mit einem der folgenden Befehle:

Produktionsmodus:
npm start
Entwicklungsmodus:
npm run dev

CMS öffnen

Nach erfolgreichem Start können Sie das CMS über folgende URLs erreichen:

Hauptwebsite: http://localhost:3001
CMS Admin: http://localhost:3001/admin.html

Login-Daten

Benutzername: admin
Passwort: hausarzt2024!
Wichtig: Ändern Sie das Passwort nach dem ersten Login!

Hilfe & Support

  • Lesen Sie die vollständige Dokumentation in CMS-README.md
  • Stellen Sie sicher, dass Port 3001 verfügbar ist
  • Bei Problemen prüfen Sie die Terminal-Ausgabe
  • Für Backup & Restore folgen Sie der Anleitung im CMS

Erweiterte Konfiguration

Für erweiterte Konfigurationen können Sie folgende Umgebungsvariablen setzen:

PORT: Ändert den Server-Port (Standard: 3001)
JWT_SECRET: Sicherheitsschlüssel für Authentifizierung
